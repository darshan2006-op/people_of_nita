# Blog Posts

This folder contains markdown blog posts for the People of NIT website.

## Structure

Each blog post is a markdown file with frontmatter metadata.

## Adding a New Blog Post

1. Create a new `.md` file in this folder with a URL-friendly filename (e.g., `my-blog-post.md`)
2. Add the required frontmatter at the top
3. Write your content in markdown below the frontmatter
4. The blog will automatically appear on the `/about` page
5. Individual blog pages will be accessible at `/blog/[filename-without-extension]`

## Markdown Support

The blog system supports:
- Headings (H1-H6)
- Paragraphs
- **Bold** and *italic* text
- Lists (ordered and unordered)
- Links
- Code blocks
- Blockquotes
- Images
- Horizontal rules
