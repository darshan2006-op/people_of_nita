# Post Images Directory

Place your blog post images here.

## Quick Start

```bash
# Add an image
cp ~/Downloads/my-image.jpg public/post_images/

# Reference in markdown frontmatter
imagePath: "/post_images/my-image.jpg"
```

## Image Guidelines

- **Format:** JPG, PNG, WebP
- **Size:** < 500KB (optimized)
- **Dimensions:** 500x500px (square recommended)
- **Naming:** kebab-case-name.jpg

See `IMAGE_GUIDE.md` in project root for complete documentation.
